# タクソノミ ジェネレータ

このディレクトリには、ローカル UADC 互換 タクソノミ ジェネレーターとそれによって使用される GL ジェネリック スキー テンプレートが含まれています。

- **xBRLGL_TaxonomyGenerator.py**LHM CSV を読み、モジュールスキーマを生成します。
EN 16931 OIM エントリーポイント、プレゼンテーションおよび定義リンクベースs、メタデータ、およびCSVスケルトン。
- **gen/**には、一般的なデータ型スキーマテンプレートが含まれています。

現在のOIMのエントリーポイントは**en16931-oim-2026-07-05.xsd**でディメンションal 定義リンクベースは**en16931-def-2026-07-05.xml**です。 タクソノミモデルと発電機内部の[**docs/DATA_MODEL.md**](../../../docs/ja/DATA_MODEL.md)を参照してください。
