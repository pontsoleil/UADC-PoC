# Figures

This directory contains image assets used by repository documentation.

## Files

- `uadc_poc_processing_flow_figure1.png` - Figure 1 from the PoC proposal,
  showing the proposed two-stage UADC processing flow from OpenPeppol invoice
  XML to hierarchical tidy data and downstream audit views.

When adding figures, prefer stable PNG files with descriptive names. Keep
intermediate render files only when they are useful for traceability.
