# References Directory

This directory contains lightweight reference material for the PoC.

## Subdirectories and Files

- `peppol_sources.md` - Notes and links for OpenPeppol / BIS Billing source
  material used by the PoC.
- `figures/` - Images extracted or prepared from proposal material for use in
  Markdown documentation.

Large licensed standards, downloaded schema archives, and temporary extraction
outputs should not be committed here. Keep those in local work areas or under
ignored generated output as appropriate.
