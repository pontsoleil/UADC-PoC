# Expected Sample Outputs

This directory contains small expected outputs for focused regression checks.

## Files

- `openpeppol_ubl_invoice_structured.csv` - Expected structured CSV output for
  the minimal OpenPeppol UBL Invoice conversion test.

Use this directory for stable, compact expected artifacts only. Larger generated
evidence should be kept under `tests/roundtrip/` for review or under ignored
`out/` for local output.
