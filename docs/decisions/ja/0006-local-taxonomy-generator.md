# 決定: タクソノミ ジェネレーターをこのリポジトリに保存する

## コンテンツ

タクソノミ 生成は UADC PoC リポジトリの一部です。**UADC PoC**のきれいなチェックアウトには、通常のタクソノミ生成と回帰テストに必要なスクリプトとテンプレートが含まれている必要があります。

## デコレーション

1. UADC 互換 タクソノミ ジェネレーターをオンにします
**tools/taxonomy/xBRLGL_TaxonomyGenerator.py**.
2. 必須の**gl-gen**テンプレートをそのまま保存
**tools/taxonomy/gen/gl-gen-2026-MM-DD.xsd**.
3. テストおよび文書のローカル発電機を使用して下さい。
4. 生成されたローカル出力として**out/taxonomy/**を扱います。
5. コミットされたソースツリーから出力されたタクソノミを生成してください。

## コンシーケンス

- **tests/test_xbrlgl_generator_uadc_lhm.py**は新しいクローンから実行できます。
- リポジトリは、PoC チェックポイントに必要なジェネレーターコードを処理します。
- Generated タクソノミ は、**out/**の下の出力を再現可能な PoC としてコミットしています
証拠。 手動で編集するのではなく、ローカルジェネレータとソースの定義から再生する必要があります。
- タクソノミ 生成の将来の変更は、**tools/taxonomy/xBRLGL_TaxonomyGenerator.py**で行われます。
