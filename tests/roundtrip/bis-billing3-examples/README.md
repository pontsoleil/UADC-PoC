# BIS Billing 3 Round-Trip Cases

This directory contains round-trip artifacts for selected BIS Billing 3 invoice
examples.

## Subdirectories

- `source_xml/` - Original BIS Billing 3 XML inputs.
- `structured_csv/` - Structured CSV generated by syntax binding conversion.
- `metadata_json/` - xBRL-CSV metadata JSON referencing the generated
  `plt-oim` taxonomy entry point.
- `roundtrip_xml/` - XML regenerated from the structured CSV.

The cases cover allowances, VAT categories, negative correction invoices, sales
order references, and tax accounting currency behavior.
