# Binding Specifications

This directory contains binding definitions used by the UADC PoC.

## Subdirectories

- `syntax/` - Source syntax bindings. The active Phase 1 binding maps UBL
  Invoice XPath expressions and selector predicates to EN 16931 LHM semantic
  paths.

## Planned Binding Types

Semantic binding definitions will be added in a later phase to project the
common hierarchical tidy data representation into downstream audit views, such
as ISO 21378 Sales/Purchase invoice outputs and AICPA ADS O2C/P2P outputs.

At this stage, the committed binding baseline is the EN 16931 UBL Invoice
syntax binding.
